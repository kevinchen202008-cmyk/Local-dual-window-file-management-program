## 💾 配置管理

### 自动保存的配置
程序自动保存到 `~/.filemanager/config.json`：
- 窗口大小和位置
- 左右面板的当前路径
- 显示偏好设置
- 排序方式

### 手动修改配置
```json
{
  "left_panel_path": "/home/user/Documents",
  "right_panel_path": "/home/user/Downloads",
  "window_width": 1400,
  "window_height": 800,
  "window_x": 100,
  "window_y": 100
}
```
