## 🎯 基本操作

### 浏览文件
- **打开文件夹**: 双击文件夹或 Enter
- **返回上级**: Backspace 或点击工具栏"⬆ 上级"
- **刷新**: F5 或点击"🔄 刷新"

### 选择文件
- **单选**: 单击文件
- **多选**: Ctrl + 单击
- **范围选**: Shift + 单击
- **全选**: Ctrl + A

### 操作文件
- **复制**: 选中 → 点击"复制" 或 Ctrl+C
- **移动**: 选中 → 点击"移动" 或 F6
- **删除**: 选中 → 按 Delete

### 搜索文件
- **打开搜索**: Ctrl+F 或 菜单→工具→搜索文件
- **输入关键词**: 支持 * 通配符
- **查看结果**: 在搜索对话框中

## 💡 常用技巧

### 快速导航
在路径输入框直接输入完整路径，按 Enter
```
例如: C:\Users\YourName\Documents
或:   D:\
或:   /home/user/Downloads
```

### 同步路径
点击工具栏"⟷ 同步路径"按钮，快速将一个面板的路径同步到另一个面板

### 文件批量操作
1. 使用 Ctrl 或 Shift 多选文件
2. 一次操作多个文件
3. 大大提高工作效率

### 查看文件属性
右键点击文件 → 选择"属性" → 查看详细信息
