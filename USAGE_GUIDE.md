# 文件管理器 - 使用指南

## 快速开始

### Windows 用户
双击 `run.bat` 文件启动程序。

### Linux/macOS 用户
打开终端，运行：
```bash
bash run.sh
```

或者直接用 Python 运行：
```bash
python main.py
```

## 界面说明

### 窗口布局
```
┌─────────────────────────────────────────┐
│          菜单栏 | 工具栏                  │
├──────────────────┬──────────────────────┤
│                  │                      │
│    左文件面板    │    右文件面板         │
│                  │                      │
├──────────────────┼──────────────────────┤
│                  │                      │
│    操作按钮栏    │                      │
└──────────────────┴──────────────────────┘
```

### 主要组件

1. **菜单栏**
   - 文件: 打开、退出
   - 编辑: 复制、剪切、删除、刷新
   - 查看: 导航、同步
   - 工具: 搜索、设置
   - 帮助: 关于

2. **工具栏**
   - ⬆ 上级: 返回上级目录
   - 🔄 刷新: 刷新当前面板
   - ⟷ 同步路径: 同步两个面板的路径

3. **文件面板**
   - 路径输入框: 直接输入路径或浏览
   - 文件列表: 显示文件和文件夹
   - 状态栏: 显示文件信息

4. **操作按钮**
   - 复制 (F5): 复制选中文件到另一面板
   - 移动 (F6): 移动选中文件到另一面板
   - 删除 (Del): 删除选中文件
   - 刷新 (F5): 刷新文件列表

## 常用操作

### 文件浏览
1. **打开文件夹**: 双击文件夹或单击后 Enter
2. **返回上级**: 点击".."或点击工具栏"⬆ 上级"或按 Backspace
3. **快速导航**: 在路径框直接输入完整路径，按 Enter

### 文件操作
1. **选择文件**:
   - 单击: 选中单个文件
   - Ctrl+单击: 多选文件
   - Shift+单击: 范围选择
   - Ctrl+A: 全选

2. **复制文件**:
   - 左面板选中文件 → 点击"复制" → 文件复制到右面板
   - 支持复制单个或多个文件夹

3. **移动文件**:
   - 选中文件 → 点击"移动" → 文件移动到另一面板
   - 原位置的文件会被删除

4. **删除文件**:
   - 选中文件 → 点击"删除" → 确认删除
   - 谨慎操作，删除后无法恢复

### 右键菜单
右键点击文件列表显示快捷菜单：
- 复制: 复制文件到另一面板
- 移动: 移动文件到另一面板
- 删除: 删除文件
- 刷新: 刷新文件列表
- 属性: 查看文件详细信息

## 进阶功能

### 搜索文件
1. 菜单 → 工具 → 搜索文件
2. 或按 Ctrl+F
3. 输入文件名 (支持 * 通配符)
4. 点击"搜索"
5. 双击结果项打开文件

### 文件属性
右键菜单 → 属性，显示：
- 文件名
- 完整路径
- 文件大小
- 修改时间
- 文件类型

### 配置保存
程序自动保存：
- 窗口大小和位置
- 左右面板的当前路径
- 用户偏好设置

## 技巧和最佳实践

### 文件同步
1. 点击工具栏"⟷ 同步路径"
2. 当前焦点面板的路径会复制到另一面板
3. 快速在相同位置进行对比操作

### 批量操作
1. 使用 Ctrl+单击 选择多个不相邻的文件
2. 使用 Shift+单击 选择范围内的所有文件
3. 批量复制/移动/删除一次完成

### 快速导航
1. 直接在路径框输入路径，按 Enter
   - 例: C:\Users\YourName\Documents
   - 例: D:\
   - 例: \\server\share

2. 点击"浏览..."按钮弹出文件夹选择对话框

## 故障排除

### 程序无法启动
1. 确保已安装 Python 3.7+
2. 运行 `pip install -r requirements.txt` 安装依赖
3. 确保 PyQt5 安装正确

### 无法访问某些文件夹
1. 检查文件夹权限
2. 以管理员身份运行程序
3. 某些系统文件夹可能受保护

### 文件操作失败
1. 检查目标目录是否存在
2. 检查磁盘空间是否充足
3. 确保没有其他程序锁定文件
4. 以管理员身份运行程序

## 配置文件

程序配置保存在：
- Windows: `C:\Users\YourName\.filemanager\config.json`
- Linux: `~/.filemanager/config.json`
- macOS: `~/.filemanager/config.json`

可以手动编辑此文件修改设置。

## 数据安全说明

⚠️ **重要提示**：
- 删除操作不可恢复
- 建议在重要操作前备份文件
- 移动大文件可能需要较长时间
- 不在文件操作中途关闭程序
