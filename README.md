# 文件管理器 - File Manager

一个功能完整的本地双窗口文件管理程序，类似于 FreeCommander，仅限本地机器使用。

## 功能特性

### 核心功能
- **双窗口布局**: 左右两个独立文件浏览面板，支持对比查看
- **文件浏览**: 快速导航文件系统，支持返回上级、进入子目录
- **文件操作**: 
  - 复制文件和文件夹
  - 移动文件和文件夹
  - 删除文件和文件夹
  - 批量操作选中的多个文件

### 用户界面
- **路径导航**:
  - 路径输入框，直接输入路径
  - 浏览按钮快速选择文件夹
  - 返回上级按钮

- **文件列表**:
  - 显示文件名称、类型、大小、修改时间
  - 支持多选操作
  - 右键菜单快速操作
  - 双击打开文件夹或文件

- **状态栏**:
  - 显示总项数、选中项数、选中文件总大小

### 快捷键
| 快捷键 | 功能 |
|-------|------|
| F5 | 刷新当前面板 |
| F6 | 移动文件 |
| Del | 删除文件 |
| Backspace | 返回上级目录 |
| Ctrl+C | 复制 |
| Ctrl+X | 剪切/移动 |
| Ctrl+Q | 退出程序 |

### 菜单选项
- **文件菜单**: 打开、退出
- **编辑菜单**: 复制、剪切、删除、刷新
- **查看菜单**: 返回上级、同步路径
- **工具菜单**: 设置
- **帮助菜单**: 关于

## 使用说明

### 基本操作
1. **浏览文件**: 点击文件夹进入，显示内容在相应面板
2. **选择文件**: 单击选择，Ctrl+点击多选，Shift+点击范围选择
3. **复制文件**: 选中文件，点击"复制"按钮或 Ctrl+C，在目标面板完成
4. **移动文件**: 选中文件，点击"移动"按钮或 Ctrl+X，在目标面板完成
5. **删除文件**: 选中文件，点击"删除"按钮或 Press Del，确认删除
6. **查看属性**: 右键选择"属性"查看文件详细信息

### 快捷键速查

| 快捷键 | 功能 | 快捷键 | 功能 |
|--------|------|--------|------|
| F5 | 刷新 | F6 | 移动 |
| Del | 删除 | Backspace | 上级 |
| Ctrl+C | 复制 | Ctrl+X | 剪切 |
| Ctrl+F | 搜索 | Ctrl+Q | 退出 |

> 更多快捷键详见 [快速参考](QUICK_REFERENCE.md)

## 技术细节

### 使用的库
- **PyQt5**: GUI 框架，提供跨平台的用户界面
- **os, shutil, pathlib**: 文件系统操作

### 架构设计
- **MVC 模式**: 分离模型、视图和控制逻辑
- **模块化设计**: 独立的文件面板和菜单模块
- **事件驱动**: 响应用户交互的信号和槽

## 系统要求

- **依赖**: PyQt5
- **内存**: 至少 512MB
