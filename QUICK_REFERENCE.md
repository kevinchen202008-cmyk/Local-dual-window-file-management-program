# 快速参考卡片

## 文件管理器 - 快速开始

### 安装和运行
```bash
# Windows - 双击运行
run.bat

# Linux/macOS
bash run.sh

# 或使用Python
python main.py
```

### 快捷键速查

| 键 | 功能 | 键 | 功能 |
|---|---|---|---|
| F5 | 刷新 | F6 | 移动 |

| Ctrl+C | 复制 | Ctrl+X | 剪切 |
| Ctrl+F | 搜索 | Ctrl+Q | 退出 |

### 鼠标操作

| 操作 | 效果 |
|---|---|
| 单击 | 选中文件 |
| Ctrl+单击 | 多选文件 |
| Shift+单击 | 范围选择 |
| 双击 | 打开文件夹 |
| 右键 | 快捷菜单 |

### 常见任务

1. **复制文件到另一面板**
   - 左面板选中 → 点击"复制" → 右面板显示副本

2. **移动文件**
   - 选中文件 → 点击"移动" → 确认

3. **删除文件**
   - 选中文件 → 按 Delete → 确认删除

4. **搜索文件**
   - 按 Ctrl+F → 输入文件名 → 点击搜索

5. **查看文件属性**
   - 右键选中文件 → 点击"属性"

### 菜单导航

**文件菜单**
- 打开 (Ctrl+O)
- 退出 (Ctrl+Q)

**编辑菜单**
- 复制 (Ctrl+C)
- 剪切 (Ctrl+X)
- 删除 (Del)
- 刷新 (Ctrl+R)

**查看菜单**
- 返回上级 (Backspace)
- 同步路径

**工具菜单**
- 搜索文件 (Ctrl+F)
- 设置

**帮助菜单**
- 关于

### 工具栏按钮

- ⬆ 上级 - 返回上级目录
- 🔄 刷新 - 刷新当前面板
- ⟷ 同步路径 - 同步两个面板路径

### 底部操作栏

- 复制 (F5) - 复制文件到对面板
- 移动 (F6) - 移动文件到对面板
- 删除 (Del) - 删除选中文件
- 刷新 (F5) - 刷新文件列表

### 技巧

**快速导航**
- 路径框直接输入完整路径，按 Enter

**批量选择**
- Ctrl+A 全选当前目录所有文件
- Shift+单击 选择范围内所有文件
- Ctrl+单击 逐个添加到选择

**文件同步**
- 点击"⟷ 同步路径"快速同步两个面板

**自动保存**
- 程序自动保存窗口位置、大小和路径

### 配置位置

所有配置保存在：
- `~/.filemanager/config.json`

可以手动编辑此文件修改设置。

### 常见问题

**Q: 如何删除多个文件？**
A: Ctrl+单击选择多个文件，然后按 Delete

**Q: 如何将文件从左面板复制到右面板？**
A: 左面板选中 → 点击"复制"按钮

**Q: 如何搜索隐藏文件？**
A: 使用菜单 → 工具 → 搜索文件

**Q: 程序数据会被保存吗？**
A: 是的，窗口设置、路径等会自动保存

**Q: 删除的文件可以恢复吗？**
A: 不可以，请谨慎操作
